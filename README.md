# AI Scientific Lab

> Personal laboratory for scientific computing, artificial intelligence and interactive visualization.

---

## Vision

AI Scientific Lab is my personal engineering and research laboratory.

Its purpose is to progressively build a deep understanding of mathematics, scientific computing, optimization, artificial intelligence and interactive visualization through implementation, experimentation and scientific reproducibility.

This repository accompanies a long-term learning journey. It is not a collection of course materials, but a place where knowledge is transformed into practical skills and reusable software.

---

## Philosophy

The laboratory is guided by a few simple principles.

- Understand before using.
- Implement before optimizing.
- Reproduce before innovating.
- Visualize to better understand.
- Document to learn.
- Build reusable components.
- Learn by creating.

The objective is not only to complete courses or obtain certificates, but to progressively develop an engineering mindset and, ultimately, a research mindset.

---

## Scientific domains

The laboratory currently focuses on the following areas:

## Scientific reproducibility

One of the long-term objectives is to reproduce scientific publications related to:

- Scientific Machine Learning
- Optimization
- Numerical Simulation
- Deep Learning
- Interactive Visualization

Each reproduction should include:

- bibliographic references;
- implementation;
- experiments;
- discussion;
- comparison with published results.

---

## Long-term goals

- Build a reusable scientific software toolbox.
- Develop interactive visualizations of mathematical algorithms.
- Explore modern AI methods for scientific computing.
- Reproduce research papers.
- Design original projects combining mathematics, AI and visualization.
